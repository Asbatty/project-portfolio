Mohr's Circle Demo Tool

A physical stress-transformation demo built with strain gauges, an HX711 amplifier, an ESP32, and a Wheatstone bridge circuit. Real-time data streams over serial to a MATLAB interface that plots Mohr's circle live.

Type Senior design project
Date Spring 2025
Role Electronics, firmware, software

Overview

The goal was to build a hands-on teaching tool for stress transformation in ME coursework. Students apply loads to a cantilever beam instrumented with strain gauges at known angles — the system reads the differential voltage across the Wheatstone bridge, amplifies it with an HX711, and streams the data over serial from an ESP32.

On the software side, a MATLAB script reads the serial port, computes the principal stresses and angle, and plots the updated Mohr's circle in real time. I also designed the schematic in KiCad after identifying GPIO pin conflicts in the original design.
