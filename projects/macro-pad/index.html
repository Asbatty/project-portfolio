Macro Pad with Display

Custom mechanical macro pad with an integrated small display. Hand-wired matrix, custom PCB, CNC-machined enclosure, and QMK firmware with display integration.

Type Personal project
Date 2024 – 2025
Role Design, fabrication, firmware

Overview

Built from scratch — starting with the enclosure CAD in Fusion 360, through PCB layout, hand-wiring the key matrix, and writing the QMK firmware layers with display output. The display shows the current layer and active macros.
