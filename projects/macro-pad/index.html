Macro Pad with Display

Custom mechanical macro pad with an integrated small display. Designed and built from scratch — enclosure, electronic schematic, & software.

Type Personal project
Date 2025 – 2026
Role Design, fabrication, firmware

Overview

Built first from Thingiverse link to gain knowledge and then rebuilt from scratch — starting with the enclosure CAD in Fusion 360, hand-wiring the key matrix, and writing the Arduino code with display output. The display shows the current profile and active macros.
